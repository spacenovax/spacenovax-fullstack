# SpaceNovaX Fullstack v2

Includes:
- Telegram Mini App UI
- SPNX Points mining
- 24h timer and claim
- Mission rewards
- Admin mission approval
- PostgreSQL-ready user database
- Future Solana wallet registration
- Admin Dashboard at /admin.html

## Render Web Service
Build Command:
npm install

Start Command:
npm start
